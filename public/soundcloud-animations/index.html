SoundCloud Logo Animations



I was messing around with this little fad of making loading animations. I had started to develop a version with pleasant rounded-bar animation sequence, when Josh Martin looked over my shoulder.

He suggested that the bars looked just like those found in the SoundCloud logo. And wouldn't it be cool to make an animation out of a logo?

It wasn't hard to build the rest of the animation sequences around the idea of sound waves and sound visualizations and old-school speaker system lights. I've placed three different versions here, all of which are built around the same genreal layout.

Each version is built entirely in releative units, so scaling the logo animation is as simple as changing one font-size: line in the CSS. All of the animations are done via CSS3 transforms and keyframes, with varying degrees of graceful degredation across browsers. The control of the animations is accomplished with a little DOM class manipulation and tiny JavaScript.

It would be cool to see something like this on page load when visiting SoundCloud tracks. It would be even cooler if the logos throughout the page, or the favicon, were animated in such a way when any track is in progress, as a visual signal to the user.

  1. Animate the logo
  2. Relate the animation to the company/product
  3. Make the animation useful to the user, codify the fluff
